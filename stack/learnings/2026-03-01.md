# Cron infrastructure + auto-updating site

**4 crons now running:**
- 8:00 AM IST: Podcast Digest (81 feeds, picks top 3)
- 8:05 AM IST: RSS Digest (13 feeds, new articles only)
- 10:30 PM IST: Daily Wrap — reads memory + GitHub commits, writes log entry to siva-builds, triggers Vercel rebuild, drafts 2 LinkedIn posts
- 3:30 AM IST: Daily OpenClaw idea

**The siva-builds data pipeline:**

`siva-builds` is a public GitHub repo that acts as the data source for this site. Every push triggers a Vercel rebuild via deploy hook. Cases live in `projects/cases.json`. Daily logs live in `daily/YYYY-MM-DD.md`. Stack learnings (this file) live in `stack/learnings/`.

The 10:30 PM cron completes the loop: work happens → memory is written → cron reads memory → commits to siva-builds → Vercel rebuilds → site updates. No manual steps.

**The gateway stability issue:**

The gateway ran as a manual process for the first few days and went down for 16 hours. The fix: `launchctl` LaunchAgent (`ai.openclaw.gateway`) ensures it restarts automatically on crash or reboot. Should have been the first thing configured.

Key restart pattern: `launchctl stop ai.openclaw.gateway && launchctl start ai.openclaw.gateway`. The `??` exit code in launchctl output is cosmetic — verify actual status with `openclaw gateway status`.

**Groq Whisper:**

Configured `whisper-large-v3-turbo` for Telegram voice transcription. Send a voice note, get a text response. Under 2 seconds. The model is significantly faster than OpenAI Whisper at equivalent quality for conversational audio.
