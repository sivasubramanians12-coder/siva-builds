## Tuesday Sales Ideation — Seven Concepts

The full Tuesday Sales theme produced a dense ideation run. Seven concepts surfaced across the day's heartbeat rotations:

1. **Sales Pipeline Digest** — morning Telegram report: deals by stage, deals at risk, overdue follow-ups. OpenClaw cron + Python reading pipeline source → structured Telegram message by 8 AM. Implementation path is identical to Order Cascade Monitor (Ops) — same architectural pattern, different data layer.

2. **Follow-Up Automation** — reads pipeline state → finds deals with no contact in N days → drafts personalised follow-up messages for rep review. Two-step: stale deal detection (threshold logic) + LLM-assisted draft generation. Pairs directly with Pipeline Digest.

3. **Prospect Intelligence Agent** — monitors signals (social, LinkedIn, real estate listings in Bangalore) for high-intent home buyers/renovators → flags to sales team. More complex: requires external data ingestion and signal scoring. Longer build, higher upside for Hillary's brand.

4. **Dealer Content Factory** — SKU + dealer context → WhatsApp-ready product descriptions, price quotes, objection-handling scripts. Runs on demand via Telegram command. LLM-heavy, low infra. Probably the fastest to ship of this set.

5. **Engagement Monitor** — tracks LinkedIn post performance (type, topic, hook style) → builds personal performance database → informs content strategy. Metadata-only, no scraping. Could run as a weekly digest.

6. **Customer Testimonial Extractor** — scans feedback forms, WhatsApp threads, emails → extracts quotable testimonials → formats for LinkedIn/Instagram. Useful for Hillary's. Needs access to the feedback pipeline first.

7. **n8n Orchestration** (separate from cron idea above) — broader evaluation of n8n as a workflow backbone for HD India automations. This is an infra/tooling concept, not a pure sales play — but surfaced in the sales rotation because the first candidate workflow is the Pipeline Digest.

## Prioritisation Read

Against the Prioritisation Matrix: **Pipeline Digest** and **Dealer Content Factory** are the immediate builds — both are high-impact, low-effort, and have a clear data source (the HD India pipeline and product catalogue). Follow-Up Automation is a natural second phase after Digest is live.

Prospect Intelligence and Testimonial Extractor require external data access — those go to backlog pending source definition. Engagement Monitor is a one-session build once the LinkedIn posting cadence stabilises (target: mid-April based on current intro series pace).
