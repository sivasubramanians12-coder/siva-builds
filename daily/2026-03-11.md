Build priority candidates for next sprint: **Prospect Intelligence Agent** (direct pipeline value) and **Post Draft Agent** (already being partially done manually — automate the blank-page step).

## Heartbeat Idea Repetition Bug

A state-tracking issue was caught during today's run: the "Idea of the Day" segment in heartbeat logs repeated the same idea multiple times across different heartbeat cycles. The root cause is no persistence between heartbeat runs — each cycle independently selects an idea with no knowledge of what was already sent today.

Fix options documented: (a) idea-state.json tracking last-sent idea ID + timestamp, (b) rotate idea by heartbeat index (not random), (c) move idea generation to a dedicated cron that fires once per day rather than per heartbeat. Option (c) is cleanest — the ideas cron already exists and can absorb this with minimal change. Needs Siva confirmation before wiring.
